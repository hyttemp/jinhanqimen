二、HTML 引入範例

SCRIPT TAG — 依序引入，不可顛倒
<!-- 1. 天文基礎層（無依賴）-->
<script src="paipan_standalone_astro.js"></script>
<!-- 2. 農曆計算層（依賴 astro）-->
<script src="paipan_standalone_lunar.js"></script>
<!-- 3. 四柱干支層（依賴 astro）-->
<script src="paipan_standalone_gz.js"></script>
<!-- 4. 奇門基礎方法層（依賴 astro + lunar + gz）-->
<script src="paipan_qimen_addon.js"></script>
<!-- 5. 日家奇門排盤層（依賴全部前四層）-->
<script src="paipan_qimen_day.js"></script>
⚠ 載入順序注意事項
所有模組共用 window.Paipan 命名空間，必須嚴格依照 1→2→3→4→5 順序載入。
lunar.js 與 gz.js（步驟2、3）均依賴 astro.js，兩者之間無相互依賴，可互換順序。
addon.js 需要 astro + lunar + gz 三層全部就緒後才可載入。
day.js 為最頂層，必須最後載入。

三、依賴鏈示意圖

paipan_standalone_astro.js （無依賴，基礎層）
├──→ paipan_standalone_lunar.js （依賴 astro）
├──→ paipan_standalone_gz.js （依賴 astro）
└──→ paipan_qimen_addon.js （依賴 astro + lunar + gz）
         └──→ paipan_qimen_day.js （依賴全部前四層，最頂層）

四、各模組 API 速查表

paipan_standalone_astro.js — 天文函式
函式 / 常數 說明 回傳 / 型別
Jdays(y,m,d,h,mi,s) 公曆轉儒略日（Julian Day Number） Number（JD）
Jtime(jd) 儒略日轉公曆日期時間物件 Object {y,m,d,h,mi,s}

VE(y) 春分點儒略日（含 T^5 項） Number（JD）
MeanJQJD(y, n) 第 n 個節氣平均儒略日（克卜勒橢圓軌道） Number（JD）
Perturbation(jd) 太陽攝動修正（ptsa/ptsb/ptsc 陣列） Number（天）
GetAdjustedJQ(y, n) 取第 n 個節氣真實儒略日（含 ΔT + 攝動 + jqXFu 修正） Number（JD）
intval(v, base) 安全整數轉換（仿 PHP intval） Number（整數）
floatval(v) 安全浮點數轉換 Number（浮點）
ctg / cdz 天干字串 / 地支字串（各10/12字） String
gz[] 六十甲子陣列（甲子～癸亥） Array[60]
jqXFu{} 節氣歷史修正表（120+ 年份，物件格式） Object

MeanNewMoon(k) 第 k 個平均朔日儒略日 Number（JD）
GetZQandSMandLunarMonthCode(y) 取得指定年份中氣、朔日及農曆月份編碼 Object {zq[], sm[], mc[]}
